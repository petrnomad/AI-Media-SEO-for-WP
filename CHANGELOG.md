# Changelog

All notable changes to this project will be documented in this file.

## [2.3.7] - 2025-01-15
### Added
- Support for Google Gemini 1.5 Pro.
- Support for Anthropic Claude 3.5 Sonnet.
- New "Advanced" prompt variant for higher quality descriptions.
- AVIF image format support with auto-conversion.

### Changed
- Updated OpenAI model to GPT-4o.
- Improved bulk processing speed and reliability.
- Refreshed admin UI with better progress tracking.

### Fixed
- Fixed an issue with token counting for large images.
- Resolved a conflict with some caching plugins.

## [1.0.0] - 2024-01-01
- Initial release.
